# Braid context

A skill and rule for building SEEK UI using the [Braid Design System](https://seek-oss.github.io/braid-design-system/).

## Contents


| Type  | Name       | Purpose                                                                              |
| ----- | ---------- | ------------------------------------------------------------------------------------ |
| Skill | `braid`    | Design system guidance — components, tones, platforms, accessibility, Backstage docs |
| Rule  | `braid-ui` | Always-on Braid UI constraints for web, native, and email UI files                   |


## Getting started

SEEKers should install using the [AI Toolkit](https://backstage.myseek.xyz/docs/default/component/seek-ai-toolkit/#seek-ai-toolkit-documentation).

```bash
# Skill
ai-toolkit install skill braid --catalog seek-oss/braid-context/.agents@master

# Rule (companion)
ai-toolkit install rule braid-ui --catalog seek-oss/braid-context/.agents@master
```
